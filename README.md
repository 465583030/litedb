### LiteDB

    一个轻便以及追求SQL性能的MYSQL客户端

#### Intro

    LiteDB 的核心设计目标是提供一个轻量级的SQL封装.
    LiteDB 不会对设计范式与Mysql本身做更多的侵入
    LiteDB 提供基本的SQL CURD封装
    LiteDB 不提供任何形式的SQLBuilder
    LiteDB 使用 `database/sql` 和 mysql驱动
